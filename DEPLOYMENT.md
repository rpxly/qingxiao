# 《青销》项目 - 部署指南

## 一、本地测试

### 1. 基本测试
1. 在浏览器中打开 `index.html` 文件
2. 检查页面是否正常加载
3. 测试所有交互功能：

2. **移动端测试**：

3. **性能测试**：
   - 使用Chrome DevTools的Performance面板分析加载性能
   - 检查图片加载时间
   - 测试页面滚动流畅度

## 二、部署方案

### 1. 国内托管平台

#### 腾讯云静态网站托管
1. **注册腾讯云账号**：访问 https://cloud.tencent.com/
2. **创建存储桶**：
   - 进入对象存储COS控制台
   - 创建新存储桶，设置为"公有读私有写"
3. **开启静态网站功能**：
   - 在存储桶配置中找到"静态网站设置"
   - 启用静态网站，设置默认首页为 `index.html`
4. **上传文件**：
   - 将 `index.html`、`css/`、`js/`、`assets/` 文件夹上传到存储桶根目录
5. **访问网站**：
   - 使用腾讯云分配的域名访问网站

#### 阿里云OSS
1. **注册阿里云账号**：访问 https://www.aliyun.com/
2. **创建存储空间**：
   - 进入OSS控制台
   - 创建新存储空间，设置为"公共读"
3. **开启静态网站托管**：
   - 在存储空间配置中找到"静态网站托管"
   - 启用静态网站，设置默认首页为 `index.html`
4. **上传文件**：
   - 将 `index.html`、`css/`、`js/`、`assets/` 文件夹上传到存储空间
5. **访问网站**：
   - 使用阿里云分配的域名访问网站

### 2. 国际托管平台

#### Netlify
1. **注册Netlify账号**：访问 https://www.netlify.com/
2. **创建新站点**：
   - 选择"Deploy manually"
   - 拖拽项目文件夹到上传区域
3. **配置构建选项**：
   - 不需要构建命令，直接部署静态文件
4. **部署完成**：
   - Netlify会自动分配一个域名
   - 可以绑定自定义域名

#### Vercel
1. **注册Vercel账号**：访问 https://vercel.com/
2. **导入项目**：
   - 选择"Import Project"
   - 选择"Upload"
3. **上传文件**：
   - 拖拽项目文件夹到上传区域
4. **部署完成**：
   - Vercel会自动分配一个域名
   - 可以绑定自定义域名

#### GitHub Pages
1. **创建GitHub仓库**：
   - 在GitHub上创建新仓库
   - 将项目文件推送到仓库
2. **开启GitHub Pages**：
   - 在仓库设置中找到"Pages"
   - 选择分支为 `main`，路径为 `/`
3. **部署完成**：
   - GitHub会分配一个 `username.github.io/repository` 域名

## 四、部署后维护

### 1. 定期更新
- **内容更新**：修改 `index.html` 中的文本内容
- **进度更新**：修改 `assets/data/progress.json` 中的进度数据
- **素材更新**：替换 `assets/images/` 中的图片文件

### 2. 性能监控
- 使用Google PageSpeed Insights分析页面性能
- 定期检查图片加载时间
- 监控页面访问速度

### 3. 问题排查

#### 常见问题及解决方案：

1. **图片不显示**：
   - 检查图片路径是否正确
   - 确保图片文件已上传
   - 检查图片格式是否支持

2. **响应式布局失效**：
   - 检查viewport meta标签是否正确
   - 检查媒体查询代码
   - 测试不同屏幕尺寸

3. **JavaScript功能异常**：
   - 检查控制台错误信息
   - 确保所有DOM元素存在
   - 测试不同浏览器

4. **页面加载缓慢**：
   - 优化图片大小
   - 减少CSS和JavaScript代码量
   - 考虑使用CDN加速
